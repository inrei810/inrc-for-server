1.インストール方法

1.inrcフォルダを開き、check.phpをメモ帳で開き、そこに記載されている指示に従い編集し、保存します。
2.inrcフォルダを、phpに対応しているサーバーに設置します。
3.inrcフォルダの中にある、admin.phpのファイル名を変更します。他人に推測されにくく、長文かつ記号、
アルファベット、数字を組み合わせたものなら安全です。

2.動作確認phpバージョン
・php5.3
・php8.0

php53以上なら動くと思います（適当）

3.管理者用の設定など
インストール方法の3で変更したphpファイルにアクセスします。
メッセージを入れステータス変更すると、INRCの利用を停止できます。
メッセージはユーザーに通知されます。
メッセージを入れ情報変更すると、INRCを使うときに警告を出せます
メッセージはユーザーに通知されます。
